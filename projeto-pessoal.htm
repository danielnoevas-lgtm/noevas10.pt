Projeto Pessoal

O teu projeto pessoal foca-se em autoconhecimento, hábitos e bem-estar. Aqui encontras ferramentas práticas para identificar o que te trava e criar um caminho de evolução.

Dicas práticas

  • Define 3 hábitos-chave para o mês.
  • Roda um diário de 5 minutos todas as noites.
  • Blinda 90 minutos semanais para aprendizagem.
  • Fala com um mentor ou amigo de confiança regularmente.
